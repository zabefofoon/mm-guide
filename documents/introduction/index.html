What is MM?

MM(Mark Up Manager) is an innovative tool that allows for more effective design and management of HTML and CSS. True to its name, 'Mark Up Manager' assists in the management of code markup, providing a GUI for intuitive and swift design.


Convenient Layout Design
MM allows easy manipulation of website layouts through GUI controls and keyboard shortcuts. Leveraging responsive attributes, you can effortlessly construct websites that adapt seamlessly to various devices, including mobile platforms.


Widget Creation Feature
You can create a variety of widgets by directly writing HTML and CSS. Build layouts and register widgets in specific areas with ease, offering a high level of customization.


Efficient CSS generation
Utilizing various properties, diverse styles are automatically formed, allowing users to easily implement their desired designs.


Workspace Configuration
Utilize the <iframe> element to effortlessly set up a personal workspace. Efficiently organize various working environments required for your projects.


Comment and Collaboration Features
Write comments to exchange opinions with other users. Strengthen team collaboration and efficiently manage feedback on projects.


MM provides an intuitive and collaboration-centric environment for both web designers and developers. Reduce the hassle of code writing and complete the web design process more efficiently.

History

MM(Markup Manager) is a service targeting key stakeholders in web development, particularly planners, markup developers, and front-end engineers. This service has been developed to streamline the web development process.


Integration of Deliverables
In the web development process, deliverables generated by each stakeholder have different formats and content. Planners create planning documents, markup developers write HTML/CSS, and front-end engineers develop projects using React or Vue. Over time, the evolving nature of each deliverable makes project maintenance challenging. MM addresses this issue by integrating each deliverable and maintaining MM's data up-to-date, providing consistent reference materials for ease of maintenance.


Conceptual Clarity
There are instances where team members involved in web development lack a common understanding. MM enhances understanding by allowing users to directly manipulate screens and providing experiences that elevate the intuition for essential web development concepts, such as 'layouts.' This facilitates effective collaboration among project team members who share the same vision and understanding.


Componentization
The majority of front-end engineers utilize technologies like React or Vue for component-based development(CBD). Reflecting this trend, MM supports planners and markup developers in conceptualizing screens in a component-oriented manner.
